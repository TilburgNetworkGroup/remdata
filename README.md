# remdata

A collection of empirical relational event history datasets for use with
the 'remverse' ecosystem.

## Installation

```r
# install.packages("devtools")
devtools::install_github("TilburgNetworkGroup/remdata")
```

## Usage

```r
library(remdata)
data(workplace2013)
str(workplace2013)
```

## Datasets

See `?remdata` for a list of all available datasets, or browse individual
help pages (e.g., `?workplace2013`).
